# 🙋‍♂️ I'm Here
Projeto mobile, cujo objetivo é cadastrar a lista de participantes de um evento.


## 💻 Tecnologias
- React Native
- TypeScript
  

## ⚙️ Funcionalidades
- Adicionar um participante na lista de presença.
- Remover um participante na lista de presença.
- Visualizar toda a lista de presença.
